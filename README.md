# 🏥 Sistema Clínica Médica - Delphi  
# 🏥 Medical Clinic Management System - Delphi

---

## 🇧🇷 Português

### 📌 Sobre o Projeto

Sistema desktop desenvolvido em **Delphi 12** para gerenciamento de clínica médica, permitindo o controle de pacientes, médicos e agendamentos.

A aplicação foi construída utilizando VCL e integração com banco de dados relacional MySQL.

---

### 🚀 Funcionalidades

- ✅ Cadastro de pacientes  
- ✅ Cadastro de médicos  
- ✅ Agendamento de consultas  
- ✅ Busca de pacientes cadastrados  
- ✅ Integração com banco de dados MySQL local  

---

### 🛠️ Tecnologias Utilizadas

- **Delphi 12**
- **VCL (Visual Component Library)**
- **MySQL (Banco de dados local)**
- **phpMyAdmin** para administração do banco

---

### 🗄️ Banco de Dados

O sistema utiliza um banco de dados MySQL local, criado e gerenciado via phpMyAdmin.

Estrutura básica:

- Tabela de Pacientes  
- Tabela de Médicos  
- Tabela de Agendamentos  

---

### 🎯 Objetivo

Aplicar conceitos de:

- Programação Orientada a Objetos (POO)
- Integração com banco de dados
- Desenvolvimento de aplicações desktop
- Organização modular do sistema

---

## 🇺🇸 English

### 📌 About the Project

Desktop application developed in **Delphi 12** for medical clinic management, allowing control of patients, doctors, and appointments.

The system was built using VCL and integrated with a relational MySQL database.

---

### 🚀 Features

- ✅ Patient registration  
- ✅ Doctor registration  
- ✅ Appointment scheduling  
- ✅ Patient search functionality  
- ✅ Local MySQL database integration  

---

### 🛠️ Technologies Used

- **Delphi 12**
- **VCL (Visual Component Library)**
- **MySQL (Local Database)**
- **phpMyAdmin** for database management  

---

### 🗄️ Database

The system uses a local MySQL database created and managed via phpMyAdmin.

Basic structure:

- Patients table  
- Doctors table  
- Appointments table

- Modular system organization
